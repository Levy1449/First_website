About this website

This website was created to help students keep their assignments managed. It provides tips and resources for effective studying. It uses a simple and clean design to ensure easy navigation and a good user experience.

Why I chose this idea

I chose this idea because throughout my time in school, I had trouble keeping track of my assignments which led to missed deadlines and increased stress.

How It Is Built

Part What it does
HTML pages Provide the content for Home, Assignment board, Study tips and About.
CSS file Keeps the layout clean and consistent without a complex design.
JavaScript file Handles a few basic tasks such as two API calls.
C# file Serves the pages and returns some data for the browser to display.

What JavaScript Does

  • Highlights the active navigation link.
  • Loads the shared header.
  • Fetches the assignment information from the server and displays it.
  • Fetches the current weather in Tel Aviv and displays it.

Server-Side Implementation

The C# backend uses .NET Minimal API. It exposes one GET route and one POST route:

  • GET /api/assignmentsget called by the JavaScript code to returns the current assignments stored on the server.
  • POST /api/assignmentspost adds a new assignment sent from the browser form.
